North Star Metrics

The metrics that determine PMF — Not vanity metrics. Metrics that indicate real value.

Primary North Star Metric

Weekly Active Decision Makers (WADM)

Definition: Number of users who use the product to make weekly decisions.

Practical proxies:

  • signals opened per week
  • boards created
  • agents run

Critical Secondary Metrics

Metric 1 — Signals Saved Per User Per Week

If < 3 → low perceived value

If > 10 → strong value

Metric 2 — Weekly Retention

If users return every week, the product has become operational.

Objective: 40% weekly retention

This is a very strong indicator of PMF.

Metric 3 — Agent Usage Rate

% of users who use at least one agent per week

If > 20% → enormous positive signal

Metric 4 — Time to First Value

Time between signup and first "aha moment"

It must be: < 5 minutes
